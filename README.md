# Mesa Turnip for Adreno 8XX (main branch)

Модифицированная сборка Mesa Turnip с улучшенной поддержкой **Adreno 8XX**

Цель проекта — улучшить стабильность и совместимость Turnip на Adreno 8XX,  
где стандартный freedreno пока не обеспечивает полноценную работу.

---

## 🔥 О проекте

Автоматический локальный сборщик драйверов adreno с выбором сборки веток.
---

## ⚠️ Что модифицировано:

- Локальная сборка с локальным sdk
- Выбор ветки для сборки
- ВУлучшена оптимизация сборки 
---

## ✔ Поддерживаемые GPU

| GPU | Статус |
|-----|--------|
| **Adreno 825** | Полностью поддерживается |
| **Adreno 829** | Полностью поддерживается |
